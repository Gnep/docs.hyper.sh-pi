Delete
------
delete a Service

### HTTP Request

`DELETE /api/v1/namespaces/default/services/{name}`

### Path Parameters

| Parameter | Description |
| --- | --- |
| name | name of the Service |

### Response

| Code | Description |
| --- | --- |
| 204 | OK |
| 400 | Bad Request |
| 404 | Not Found |
| 500 | Server Error |
