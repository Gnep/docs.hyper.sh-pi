How It Works
-----------------------------
_Pi_ provides the standard, upstream Kubernetes Pod APIs, for you to run containers. To do it, the steps are:

- Build Docker images
- Push the images to a registry
- Define your pod spec
- Call _Pi_ API to launch containers
